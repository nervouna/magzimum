= Magzimum =
* by the Theme Palace

== Description ==
Magzimum is a  simple, clean and responsive WordPress theme designed to suit the need of Magazine, Blog and News sites. With Magzimum, we've leveraged the built-in WordPress customizer, so it's now easier than ever to make your website your own. Easily add a custom logo, fav icon, web clip icon, social icons, background, custom menu, slider, featured content, breaking news, pagination, bradcrumb and more.
